Υλοποίηση διαδικτυακής πανεπιστημιακής πλατφόρμας όπου οι
καθηγητές μπορούν να δημοσιεύουν ερευνητικά θέματα και οι
φοιτητές να εκδηλώνουν ενδιαφέρον και να υποβάλλουν τις
αιτήσεις διατριβής τους. Χρήση σύγχρονων full stack
τεχνολογιών: Backend εφαρμογή υλοποιημένη σε python με
διαθέσιμο Rest API, NoSQL Datastore χρησιμοποιώντας MongoDB
και UI με χρήση React. Τα τμήματα της εφαρμογής υλοποιήθηκαν
ως πλήρως containerized χρησιμοποιώντας docker και
docker-compose
